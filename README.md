# Sethika Constructora - Catálogo Digital

Catálogo web de equipos pesados en renta. Desplegado en Vercel.

## Stack
- HTML5 + CSS3 + Vanilla JS
- Firebase (Firestore + Storage)
- Vercel (hosting)

## Actualizar contenido
El cliente puede actualizar equipos, fotos y precios desde el panel admin integrado en el catálogo.

## Deploy
Cada push a `main` activa un deploy automático en Vercel.
